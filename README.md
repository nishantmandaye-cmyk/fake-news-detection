# 🔍 Fake News Detection using Machine Learning

A machine learning project that detects fake news articles using NLP and multiple classification algorithms.

## 🔗 Live Demo
[Click here to try the app](#) ← update this after Streamlit deployment

## 📊 Results

| Model               | Accuracy | ROC-AUC |
|---------------------|----------|---------|
| Passive Aggressive  | 99.1%    | 99.7%   |
| Linear SVC          | 99.2%    | 99.8%   |
| Logistic Regression | 98.7%    | 99.6%   |
| Random Forest       | 99.0%    | 99.6%   |
| Multinomial NB      | 94.3%    | 98.2%   |

## 🛠 Tech Stack
- Python, Scikit-learn, Pandas, NumPy
- TF-IDF Vectorization
- Streamlit (web app)

## 🚀 Run Locally
pip install -r requirements.txt
python fake_news_complete.py

## 📁 Dataset
Kaggle — Fake and Real News Dataset (44,000 articles)
